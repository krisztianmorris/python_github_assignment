# Study Time Tracker

This Python program asks the user how many hours they studied today and estimates how many hours they will study in a full week. It helps users understand their study habits and weekly progress.

## How to Run
Run the program from a terminal using:

python app.py  
# or  
python3 app.py

## Example Output

How many hours did you study today? 2  
You are on track to study 14 hours this week.
